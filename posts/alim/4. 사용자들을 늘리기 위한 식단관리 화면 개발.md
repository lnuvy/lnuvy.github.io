#### 생산관리, 자원 관리 이외의 가벼운 프로젝트도 맡아보았습니다

<code style='font-size: 100%; line-height: 1.5'>
입사 말미에 맡았던 '식단관리'는 에이림이 실제 유저층을 늘려 투자를 유치하려는 목적으로 만든 파트였습니다.
기존에는 임원진과 생산/영업 관리자만 쓰던 프로그램이었는데, 이 기능 하나로 이용률이 <u>15% → 80% 수준까지 증가</u>했습니다.
<br />

아직 기본기가 부족했지만, 사용자를 직접 관찰하고 그들의 불편함에서 출발하는 방식으로 작업했습니다.

1. 식사 여부 체크는 모바일 유입이 많을 것으로 예상하여 모바일 최적화 화면으로 제작
2. 실제 식사인원 체크 화면은 태블릿에 맞춰 설계하고, 저장 버튼 없이 자동 반영
3. 식당 관리자의 반복 입력 피로를 줄이기 위해 input focus 최적화/자동화
4. 기존 ERP 화면과 일관된 디자인 컨셉 유지

<br />
기술 수준이 높지 않아도, 사용자를 제대로 이해하면 임팩트 있는 결과를 만들 수 있다는 걸 처음으로 체감한 경험이었습니다.
